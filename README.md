# Bravo Browser
Bravo is based on AOSP Browser and Chromium, and is aim to providing the ultimate browsing experience to users.

## Version
[1.1.1.912](/versions/Bravo_1.1.1.912.apk)

* No images
* Full screen
* Bookmarks
* Download manager
